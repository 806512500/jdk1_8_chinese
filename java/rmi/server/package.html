提供支持 RMI 服务器端的类和接口。一组类由 rmic 存根编译器生成的存根和骨架使用。另一组类实现了 RMI 传输协议和 HTTP 隧道。

已弃用：HTTP 隧道。 HTTP 隧道机制已被弃用。请参阅 {@link java.rmi.server.RMISocketFactory} 以获取更多信息。

已弃用：骨架和静态存根。 骨架和静态生成的存根已被弃用。这包括此包中需要使用骨架或静态存根的 API，它们的运行时支持，以及使用 {@code rmic} 存根编译器生成它们的过程。在平台的未来版本中可能会移除对骨架和静态存根的支持。骨架是不必要的，因为服务器端的方法调度由 RMI 运行时直接处理。静态生成的存根也是不必要的，因为存根是使用 {@link java.lang.reflect.Proxy Proxy} 对象动态生成的。请参阅 {@link java.rmi.server.UnicastRemoteObject UnicastRemoteObject} 以获取有关动态存根生成的信息。生成骨架和静态存根通常作为应用程序构建过程的一部分，通过调用 {@code rmic} 工具来完成。这是不必要的，可以省略对 {@code rmic} 的调用。 @since JDK1.1
