{@link java.util.regex.Pattern} 类的实例表示一个以类似于 Perl 的语法在字符串形式中指定的正则表达式。

{@link java.util.regex.Matcher} 类的实例用于将字符序列与给定模式进行匹配。通过 {@link java.lang.CharSequence} 接口向匹配器提供输入，以支持从多种输入源进行匹配。

除非另有说明，否则将 null 参数传递给此包中任何类或接口的方法将导致抛出 {@link java.lang.NullPointerException NullPointerException}。
